NOSSA VISÃO

Nosso projeto foi desenvolvido com o intuito de conectar usuários que possuem equipamentos eletrônicos antigos ou quebrados, para realizarem um descarte consciente. Rebendo benefícios por isso e o mais importante, ajudando o meio ambiente!

PRODUTOS

Os produtos que atualmente são possíveis de realizar o descarte são:


* Notebook * Tablet * TV * Celular


Ao descartar um produto em uma loja cadastrada, você não só estará fazendo um bem para você mas para todo o meio ambiente, um produto eletrônico jogado de forma indevida no lixo causa grandes estragos a natureza.
